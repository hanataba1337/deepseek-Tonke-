# DeepSeek Token Monitor 🖥️📊

> 💸 **你的每一分钱都花在哪了？直接在桌面上告诉你。**

通过 DeepSeek 官方 API 查询余额，跟踪余额变化自动计算实际消耗的桌面悬浮窗工具。

## 快速使用

**直接运行（无需安装 Python）：**

```
双击 DeepSeekMonitor.exe
```

**或者通过 Python 源码运行：**

```bash
pip install -r requirements.txt
python main.py
```

启动后在悬浮窗底部点 **`+ API`**，输入你的 DeepSeek API Key 按回车即可。

## 功能特性

- **余额跟踪** — 通过官方 API 获取余额，比较余额变化自动算出今日/本月实际消耗
- **无需数据库** — 不依赖任何第三方日志，直接读 DeepSeek 官方接口
- **月度预算** — 自行设定预算上限，实时显示已用/剩余，低于 20% 变色提醒
- **可拖拽** — 桌面常驻半透明窗口，拖拽标题栏任意摆放
- **绿色免安装** — 单文件 exe，双击即用

## 工作原理

```
DeepSeek Monitor
      ↓
  GET api.deepseek.com/user/balance
      ↓
  记录余额快照 → 下次启动比对差额 = 实际消耗
```

每次启动时记录当前余额，下次启动时比较余额变化，差值即为实际花费。

> 第一次启动显示 ¥0（建立快照），正常使用后重开即显示真实消耗。

## 常见问题

**Q: 一定要填 API Key 吗？**
A: 是的，本工具通过 DeepSeek 官方 API 获取余额，需要 API Key。

**Q: API Key 安全吗？**
A: 存在本地的 `settings.json` 中，仅用于查询余额，不会上传或泄露。

**Q: 为什么今天费用显示 ¥0？**
A: 第一次启动时建立余额快照，下次启动才能比较出差额。正常使用后重开即可。

**Q: 会影响网络性能吗？**
A: 不会。每 2 秒查询一次余额，数据量极小。

**Q: 预算怎么改？**
A: 直接在悬浮窗的预算输入框里输入数字，按回车保存。
